🚀 My First Docker Project

This project showcases my hands-on experience with Docker by building, pushing, and verifying a Docker image using Docker Hub.

---

📌 Project Overview

In this project, I created a simple Python application and containerized it using Docker.
Then, I pushed the Docker image to Docker Hub and verified it by pulling and running the container.

This project helped me understand real-world DevOps workflows.

---

🔧 What I Did

- ✅ Built a Docker image from scratch
- ✅ Created a Python app inside a container
- ✅ Fixed Docker push permission errors
- ✅ Successfully pushed image to Docker Hub
- ✅ Verified the image using "docker pull"
